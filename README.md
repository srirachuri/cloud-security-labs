This repository contains my hands-on learning and practice of Bash scripting focused on:

- Linux fundamentals
- Automation
- System monitoring
- Cloud & security use cases

This project is part of my journey towards becoming a **Cloud Security Engineer**.

## 📚 Topics Covered

### 🔹 Day 1–5: Linux Basics
- Navigation (`pwd`, `ls`, `cd`)
- File operations (`cp`, `mv`, `rm`)
- Viewing logs (`cat`, `less`, `tail`)
- Text processing (`grep`, `wc`, `sort`, `sed`)
- File permissions (`chmod`, `chown`, `sudo`)

### 🔹 Day 6–10: System & Automation
- Process monitoring (`ps`, `top`)
- Disk usage (`df`, `du`)
- Writing Bash scripts
- Conditions (`if/else`)
- Loops (`for`, `while`)
- File integrity checks (`sha256sum`)
- Network checks (`ss`)

### 🔹 Day 11–14: Real-World Projects
- 🔐 SSH Log Monitoring Script  
- 📦 Backup Automation Script  
- 🌐 Port Scanner  
- ☁️ AWS S3 File Upload  
- 🔍 Security Audits & Checks
